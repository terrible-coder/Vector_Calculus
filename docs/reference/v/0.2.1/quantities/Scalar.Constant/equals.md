## Function equals

Checks whether two [Scalar.Constant](reference/v/0.2.1/quantities/Scalar.Constant) objects
hold the same value.

### Parameters

* **that**: [Scalar.Constant](reference/v/0.2.1/quantities/Scalar.Constant)

### Returns
