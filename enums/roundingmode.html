Enumeration RoundingMode

Specifies a rounding behaviour for numerical operations on BigNum which are capable of discarding some precision. This is based on the Java implementation of rounding behaviour. Each rounding mode indicates how the least significant digit of the rounded result is to be calculated. If fewer digits are returned than the original number then the discarded digits are called the discarded fraction.

UNNECESSARY:
